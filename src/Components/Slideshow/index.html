slideshow

Basic Usage

A slideshow is an abstract component that adds a slide effect between components. The height of the slideshow will automatically grow to the next slide. Since each element must be different, or a key must be defined for Vue transitions, the component will automatically add the slide index as a key if a key isn't define. The component handles the differing animations for going backwards and forwards.
